# CS50P – Introduction to Programming with Python

## About the Project

This repository documents my personal learning journey through **CS50P – Introduction to Programming with Python**, a course by **Harvard University**.  
Here, I share my **solutions**, **notes**, and **experiments** as I learn to program using **Python**, focusing on **problem-solving**, **clean code**, and **computational thinking**.  
The goal is to develop a solid foundation in programming logic and apply it to real-world challenges.

## Key Features

* Learn **Python syntax**, **control flow**, and **functions**.  
* Work with **strings**, **lists**, **dictionaries**, and **sets**.  
* Explore **file handling**, **regular expressions**, and **object-oriented programming (OOP)**.  
* Write and run **unit tests** with `pytest`.  
* Solve **real-world inspired exercises** that strengthen logical thinking.  
* Build a **final project** integrating all learned concepts.
